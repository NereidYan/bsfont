# 練習 - 建立 jQuery 應用程式

1. 請在 [https://jquery.com/](https://jquery.com/) 下載 jquery.min.js 檔案

1. 請在 `<body>` 底端引用 jquery.min.js 檔案

    ``` html
    <script src="jquery-3.3.1.js"></script>
    ```

1. 請在 jquery 檔案之後, 撰寫下列語法

    ``` html
    <script>
        $(function() {
            $('.btn').on('click', function () {
                alert('Hello jQuery!');
            });
        });
    </script>
    ```

1. 開啟瀏覽器, 檢視網頁
